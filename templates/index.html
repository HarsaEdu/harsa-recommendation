Course Recommendation API Documentation

Use the API endpoint /api/recommend to get course recommendations for a customer.

Send a POST request with JSON payload: {"user_id": 1}

Example Response:

{
    "user_id": 1,
    "recommendations": [
        {"course_id": 5, "predicted_rating": 4.2, "is_in_interest_categories": true},
        {"course_id": 3, "predicted_rating": 3.9, "is_in_interest_categories": true},
        {"course_id": 1, "predicted_rating": 3.5, "is_in_interest_categories": true},
        {"course_id": 2, "predicted_rating": 3.1, "is_in_interest_categories": true},
        {"course_id": 4, "predicted_rating": 3.8, "is_in_interest_categories": false}
    ]
}
